## Quick start

1. Read `bios.md` — the main deliverable
2. Pick one of each prompt's 10 options (or use my recommended combination at the bottom)
3. **Verify your account on day 1** — +93% match rate, free, takes 30 seconds. The single biggest lever in any app.
4. Fill all 6 photo slots, all 3 prompts, all interest badges (completeness = #1 algorithmic input)
5. Set up an Opening Move (default question for matches)
6. Don't go live until photos + bio + prompts are all set
