## Key Stats

- **Opening Moves** (women set the first question) boosted quality conversations **77%**
- Women must message within **24 hours of matching** or the match expires — your profile must give them an obvious opener
- Verification (Photo Verification badge) → **+93% increase in match rate** (Bumble's biggest single algorithmic lever)
- Profile completeness is the **#1 ranked algorithmic input**

## Algorithm

- Inputs (in order of weight): **profile completeness** > photo quality > verification > response velocity
- The algorithm rewards **fast replies and high reply rate** — slow responders get suppressed
- Verification (the Photo Verification badge) is the highest-leverage free win
- Compliments feature in 2026: women can send a message tied to a specific prompt before matching → **opinion-heavy, polarizing prompts attract Compliments**

## Audience

- Skews intentional / relationship-oriented vs. Tinder
- Decision speed: ~10–20 seconds per profile (slower than Tinder, faster than Hinge)
- Women are the active party — **your profile must give her something to open with**
- Strong 30+ population; well-suited to a 41M serious dater
- More polished norms than Hinge — slightly more appearance-conscious audience

## Unique Features

- **Women message first** (heterosexual matches) — your profile is a conversation-starter generator
- **Opening Moves** — women can set a default question for matches to answer
- **Compliments** — message tied to a specific prompt, sendable pre-match (huge for opinion-bait prompts)
- **Bumble BFF / Bumble Bizz** — mode switching for friendship/networking
- **Interest badges** — small but additive signal
- **24-hour match expiration** — emphasizes urgency in profile design
