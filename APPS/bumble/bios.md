# Bumble — Bio + Prompts

A complete Bumble profile package: **one substantive bio** (≤300 chars) + **3 prompts × 10 answer options each** (30 prompt answers in total).

All bios and prompts respect Bumble's hard limits:
- Bio: ≤300 chars
- Each prompt: ≤120 chars

## Why Bumble is structured differently

Bumble is the **opener-bait app**. Women send the first message; matches expire in 24 hours; your profile's job is to give her **3–5 distinct hooks** she can grab to start a conversation.

Two things that change the game in 2026:
- **Verification = +93% match rate** (Bumble's biggest free lever)
- **Compliments feature** lets women send a message tied to a specific prompt *before* matching → opinion-heavy prompts now attract messages directly

The bio + prompts strategy below plants multiple hooks of different types: an identity-thesis bio, a comedy-taste prompt, an opinion-bait prompt (Compliments-magnet), and a relationship-vision prompt.

### How this bio is built for Bumble specifically

| Line | What it does |
|------|--------------|
| *"Pilot → 3D animation → video games — curiosity is a polite word for it."* | The arrow visual reads fast in Bumble's ~10-second scan window. "Curiosity is a polite word for it" is the self-aware line that humanizes the impressive-sounding career arc. |
| *"Songs with my band, pies on Sundays, miniature TV sets I made in my apartment."* | Three concrete openers — each is a hook a woman can message about. Not a list of hobbies; a list of *concrete things you've actually made.* The "miniature TV sets" beat is the *what?* moment that gets Compliments. |
| *"Bad at pickleball, getting better."* | The warmth + humility line. Critical for Bumble — proves you're not curating a polished image. Self-aware athletic confidence reads as *much* warmer than gym-bro confidence. |
| *"Looking for someone whose curiosities go past midnight."* | The relationship-vision close. Filters for depth and conversation, but framed as desire rather than declaration. Easy to react to. |

### Why this works for a 41M serious dater on Bumble

- **No demographic stat-dump** — age, location, kid status all handled by structured fields above the bio
- **Range signal in 30 words** — pilot, animator, game artist, musician, baker, maker, athlete — without listing them as a tag cloud
- **3 distinct conversation hooks** (career arrow, miniature TV sets, pickleball) so the woman has 3 options for what to message about
- **The "going to bed past midnight" close** signals depth without saying "looking for serious relationship"

### Why this combination

- **Four hooks, four different surfaces.** Bio gives "miniature TV sets" + "pickleball" + "midnight curiosities" hooks; prompts add a comedy-taste hook, an opinion-bait hook, and a relationship-vision hook. **Six total openers in one profile** — Bumble women have to actively *avoid* finding something to message about.
- **One Compliments-magnet** (Prompt 2). The In-N-Out hot take is funny, polarizing, and easy to react to → pre-match Compliment likely.
- **Voice consistency.** All four pieces sound like the same person — confident, curious, slightly opinionated, warm.
- **The "going to bed past midnight" line + "argue about an album for 3 hours" = the same person.** They reinforce. The profile reads as a unified vision: someone who wants depth + conversation + a partner who's actually *engaged* with life.

### Alt picks if you want different vibes

- **More humor-forward:** Swap Prompt 1 for option 1.6 ("Are these credit cards working out for you?")
- **More earnest:** Swap Prompt 3 for option 3.3 ("midnight thoughts and you actually share them")

# Implementation notes

- **Verify the account on day 1** — +93% match rate. The single biggest lever in any dating app, full stop.
- **Use all 6 photo slots, all 3 prompts, all interest badges** — completeness is the #1 algorithmic input
- **Reply within hours** — Bumble heavily weights response velocity
- **Don't iterate the bio for 2–3 weeks** once live; let the algorithm calibrate
- **Set up an Opening Move** (the default question matches answer) — boosts quality conversations 77%
- A/B test prompts, not the bio. Swap one prompt at a time after 2 weeks if underperforming.
