# Tinder — Everything Tinder-Related

This folder consolidates research, strategy, photo guidance, and bio + prompts for Tinder. Mirrors the `APPS/hinge/` and `APPS/facebook-dating/` structure.

## Files

| File | What's in it |
|------|--------------|
| `platform_facts.md` | 9-photo grid, 500-char bio (15–45 word optimal), 124-char prompts, available prompt list |
| `research.md` | Tinder-specific findings: 4x match rate with bios, 3–7s decision window, 3-Trait Framework, humor effect |
| `strategy.md` | Tactical playbook — tone, bio architecture, prompt strategy, "never married" framing |
| `photo_strategy.md` | 6-photo Tinder framework — slot 1 = 80% of decision; reference lineup |
| `bios.md` | **Bio + 30 prompt answer options + my pick (main deliverable)** |
| `profile_template.md` | Original profile-stub template |

## Quick start

1. Read `bios.md` — the main deliverable
2. Pick one of each prompt's 10 options (or use my recommended combination at the bottom)
3. Verify your account (+30% match rate, free)
4. Use `photo_strategy.md` for the photo lineup
5. Don't go live until photos + bio + prompts are all set — first 24–72 hours are weighted

## My pick (TL;DR)

**Bio:**
> Career went: pilot → 3D animation → video games. Songs, pies, and miniature TV sets on the side. Will explain why a Stradivarius is from a volcano if you don't stop me.

**Prompts:**
- Two truths and a lie: *"Got my pilot's license before my driver's license. Bake every Sunday. Once won a karaoke competition."*
- We'll get along if: *"…you have your own version of weird and you're not precious about it. Bonus points if you make something."*
- Change my mind about: *"…we're living in the greatest era of music ever made. I've been writing this argument in my head for years."*

## Big levers, ranked

1. **Verify the account** — +30% match rate
2. **Slot 1 photo** — 80% of the decision; don't compromise
3. **Bio under 200 chars** with the 3-Trait Framework
4. **Don't be promiscuous swiping** — selectivity is rewarded
5. **First 24–72 hours after launch** are extra weighted; go live with everything optimized
