## Quick start

If you're picking back up here cold:

1. Read `strategy.md` to remember the tactical model
2. Read `profile_drafts.md` (start with the "Proven tips" tier list at the top)
3. Pick one of the three vibes; pick one option from each of the three prompts
4. Use the lineup in `photo_lineup.md` for photos
5. Verify the account, fill all 6 photo slots, fill all 3 prompt slots

## Big levers, ranked

1. **Verify the account** — +30–93% match rate (free, takes 60 seconds)
2. **Fill all 6 photo slots and all 3 prompts** — +73% quality matches just from completion
3. **Add a voice prompt** — +32% date conversion, <5% adoption
4. **Pair `chat_03` (Pop-Tart) with a curiosity prompt** — biggest comment-rate lever in your inventory
5. **State "never married, no kids, want both" inside one prompt answer** — neutralizes the #1 first-question on every match
