## Quick start

If you're picking back up here cold:

1. Read `bios.md` — the main deliverable. Five fully-drafted bios + my pick + comparison table
2. Read `strategy.md` for the tactical model (helps you tweak the bio if needed)
3. Pick a bio (or A/B test two)
4. Use `photo_strategy.md` to choose your 6 photos
5. Connect Instagram, join 2–3 real Facebook Groups, complete every profile field, then go live

## Big levers, ranked

1. **Complete every profile field** — the algorithm weights completeness highly and FB Dating is free, so this is pure upside
2. **Connect Instagram** — adds a context layer beyond your 6 photos at zero effort
3. **Join 2–3 real Facebook Groups** before going live — directly expands your matching pool via the shared-groups signal
4. **Use Bio 5** (or your chosen bio from `bios.md`) — natural tone, locally rooted, range-signaling
5. **Use the Secret Crush feature** — uniquely powerful: up to 9 existing FB friends; only mutual selections see each other

## My bio pick (TL;DR)

**Bio 5 — Active / Wisconsin Energy.** It's the strongest fit for FB Dating's audience and algorithm specifically. Full reasoning in `bios.md`. Runner-up is Bio 3 (Maker / Curious) if you want to filter harder for creative-leaning matches.
